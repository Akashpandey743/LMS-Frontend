# Learning management system frontend

## setup instruction
1. clone the project

```
  git clone https://github.com/Akashpandey743/LMS-Frontend.git

  ```
  2. move into the directory

  ```
     cd LMS-Frontend

   ```
  3. Install dependencies

  ```
     npm i 
  
 ```

 4. Run the server

 ```
   npm run dev

  ```
